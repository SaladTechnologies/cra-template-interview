# @saladtechnologies-oss/cra-template-interview

This is an interview project template based on the official TypeScript template for [Create React App](https://github.com/facebook/create-react-app).

To use this template, add `--template @saladtechnologies-oss/interview` when creating a new app.

For example:

```sh
npx create-react-app my-app --template @saladtechnologies-oss/interview
```

For more information, please refer to:

- [Getting Started](https://create-react-app.dev/docs/getting-started) – How to create a new app.
- [User Guide](https://create-react-app.dev) – How to develop apps bootstrapped with Create React App.
